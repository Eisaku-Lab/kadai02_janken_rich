# ①課題名
カイジの限定じゃんけん（リッチ版）

## ②課題内容（どんな作品か）
漫画「賭博黙示録カイジ」に登場する“限定じゃんけん“を簡易に再現したゲーム。
【ルール】
・プレイヤーとコンピュータは各手（グー・チョキ・パー）を2回ずつ使用可能
・双方が★を2つ保有してスタート
・じゃんけんに負けた方は★を1つ失う
・★が0になった方の敗北

## ③アプリのデプロイURL

  ※実際のURLに置き換えてください

## ⑤工夫した点・こだわった点
### アプリの制作意図
- 元々のじゃんけん課題にゲーム性を加えてよりおもしろくする
- 実際のじゃんけんと比較して不足していた要素
- ① 報酬性：勝敗に意味を持たせる仕組み
- ② 連続性：過去の選択と結果が次の選択に影響を与える構造
- これらを実現するルールで、デジタル表現に適した「カイジの限定じゃんけん」を採用

### 工夫した点
#### デザイン面
- 視覚的フィードバック
  - 残り手札を●（ドット）で表示し使用済みは灰色に変化
  - 星の表示は金色の背景で強調

- アニメーション効果
  - ステータス・手札にカーソルを合わせた際の浮き上がりフェクト
  - コンピュータの手の"？"の浮遊

- カイジ風のカラーリング

#### 機能面
- 使いやすいUI
  - 使用済みの手・残りの手札は自動的にグレーアウトして選択不可に
  - 絵文字（👤🤖）を活用して直感的に操作可能

## ⑥難しかった点・次回トライしたいこと（又は機能）
### 難しかった点
- data属性を使ったドットの要素更新
  - `$('[data-player="player"][data-hand="gu"]')` のような属性セレクタの使い方

- CSS設計
  - カイジのイメージカラーを再現しつつ、視認性を保つ配色のバランス調整
  - アニメーションのタイミング調整

### 次回トライしたいこと
- 演出の強化
  - 勝敗によって背景やBGMが変わる演出を追加
  - コンピュータの手を選ぶ前に「じゃんけんぽん！」のカウントダウン演出

- ゲーム性の向上
  - 2人対戦モード（コンピュータ対戦ではなく友達と遊べる）
